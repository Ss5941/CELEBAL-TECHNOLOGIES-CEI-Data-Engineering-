# Pandas Data Cleaning Project

## Description
This project demonstrates basic data analysis and data cleaning using Python Pandas.

The project includes:
- Loading a CSV dataset
- Exploring the dataset
- Handling missing values
- Removing duplicate records
- Filtering rows and selecting columns
- Creating a derived column (total_amount)
- Saving the cleaned dataset as a new CSV file

## Technologies Used
- Python
- Pandas
- Jupyter Notebook
